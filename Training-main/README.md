# Java Programming Practice

This repository contains Java programs created while learning core programming concepts.

The programs are organized into folders based on different Java topics such as arrays, methods, object-oriented programming, and pattern problems.

## Topics Covered

- Arrays
- Basic Problems
- Classes
- Methods
- Object-Oriented Programming (OOPS)
- Strings
- Pattern Programs

## Repository Structure

java_course
│
├── Arrays
├── Basic_prblms
├── Class
├── Methods
├── OOPS
├── Strings
└── patterns

Each folder contains Java programs related to that topic.

## Learning Objectives

This repository focuses on practicing and understanding:

- Java syntax and structure
- Problem-solving using Java
- Object-Oriented Programming concepts
- Working with arrays and strings
- Pattern programming

## Technologies Used

- Java
- IntelliJ / Eclipse / VS Code (Java extensions)
